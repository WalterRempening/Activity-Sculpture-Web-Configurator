Activity Sculpture Web Configurator

Hello I am Walter Rempening Diaz. This project is part of my Master's thesis in the Mediainformatics Department at the Munich Unversity. I am researching on how to build a user friendly web configurator for activity sculptures created from personal activity data. The activity data is gathered using an activity tracker. The configurator can be used by signing in with your Withings® account.
After you login, the configurator will present you an overview of your data. When you have familiarized yourself with the data, go ahead and create a new sculpture. When you are happy with your sculpture please save it, you will then be invited to take part in a short survey.

Terms of Use

The goal of this configurator is to find out how do you interact with the configurator and how easy is it for you to use the system. The configurator stores your activity data in a private server owned by me. The information stored will only be used for research purposes. The data will be anonymized and it will not be available to third parties. After the thesis is done, all data will be erased from the server.
This project is by no means affiliated to Withings®. The Withings API is only used for gathering user data.
